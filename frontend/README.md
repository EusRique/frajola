# frontend

## Configuração do projeto
```
npm install
```

### Compila e recarrega para desenvolvimento
```
npm run serve

localhost:8080/
```

### Compila e minifica para produção
```
npm run build
```

### Lints e correção de arquivos
```
npm run lint
```
### Serviços utilizados ao executar o docker-compose

- VueJs versão 2.6
- NodeJs versão 16.13.2
- NPM versão 8.1.2
